## 💸 SpendWise – Smart Expense Tracker

SpendWise is a modern web application that helps users track, manage, and analyze their expenses efficiently.
It provides insights into spending habits and helps users make smarter financial decisions.

### 🚀 Features
- 📊 Track daily expenses and income
- 📈 Visualize spending with charts and analytics
- 🧾 Categorize transactions (Food, Travel, Bills, etc.)
- 🔍 Filter and search expenses
- 💡 Smart insights for better financial planning
- 🔐 Secure user authentication
## 🛠️ Tech Stack

### 🎨 Frontend
- React.js
- Tailwind CSS

### ⚙️ Backend
- Spring Boot

### 🗄️ Database
- MySQL

## ⚙️ Installation & Setup

### 1. Frontend setup
```bash
cd frontend
npm install
npm start
```

### 2. Backend setup
```bash
cd backend
mvn spring-boot:run
```


🌐 API Configuration

Make sure backend is running on:
```
http://localhost:8080
```
